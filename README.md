# Fraud Message / Email Detector

## 📌 Project Description
This project detects whether a message or email is Spam or Not Spam using Machine Learning and Flask web application.

## ⚙️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NLTK
- HTML
- CSS

## 🚀 Features
- Spam detection system
- Machine learning model (Naive Bayes / Logistic Regression)
- Web interface using Flask
- Input validation and clean UI

## 📂 Dataset
- SMS Spam Collection Dataset
